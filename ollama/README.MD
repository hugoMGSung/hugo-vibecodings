#### 터미널 실행

- 설치 및 버전확인

    ```powershell
    > ollama --version
    ollama version is 0.21.1
    ```

- Codex 설치

    ```powershell
    > ollama launch codex
    Error: codex is not installed, install from https://developers.openai.com/codex/cli/
    ```
